# Proxy Scraper

This repository contains regularly updated proxy lists, automatically refreshed with fresh proxies from multiple sources.

## Features
-  Automatically updates proxy lists
-  Supports multiple proxy protocols: HTTP, SOCKS4, and SOCKS5
-  Automatic deduplication of proxies
-  Tracks and displays proxy counts
-  Fully automated with GitHub Actions
